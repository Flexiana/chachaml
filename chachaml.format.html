chachaml.format

Shared formatting utilities for human-readable output.

Used by both chachaml.repl (terminal) and chachaml.ui.views (HTML). Centralised here to avoid duplication.

fmt-duration

(fmt-duration start end)

Format the elapsed time between two millisecond-epoch timestamps.

view source

fmt-instant

(fmt-instant ms)

Format a millisecond-epoch timestamp as yyyy-MM-dd HH:mm:ss.

view source

last-metric-value

(last-metric-value rows)

Given a seq of metric rows for one key (sorted by step), return the value of the last step. Used for metric summaries.

view source

metric-summary

(metric-summary metrics)

Extract the latest value per metric key from a seq of metric rows. Returns {:accuracy 0.94 :loss 0.12}.
